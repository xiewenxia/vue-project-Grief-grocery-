#### 使用方式

* 构建生产代码 `npm run build`

#### 项目第一天注意事项
* 如果轮播图无法出现，确保index.html上的doctype删除
* 如果项目中出现一些奇怪无法理解的报错,  类似es6.tostring...
    - 重新解压node_modules包
* 注重vue代码编写，css/html可以简化一点
